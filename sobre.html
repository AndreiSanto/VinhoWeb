Relatório do trabalho

Este relatório tem como objetivo descrever sucintamente as atividades ocorridas neste trabalho, de forma de fácil compreendimento do leitor.

As atividades realizadas neste trabalho foram a implementação de um sistema de distribuidora de bebidas online no formato de ecommerce. O sistema foi implementado usando a linguagem de programação PHP, usando o banco de dados relacional com a linguagem SQL.Também foi utilizado o servidor Web de codigo aberto XAMPP, onde criamos um servidor em nossa propria máquina para realizar os testes. Para a interface, foi usada a linguagem de marcação de hipertexto (HTML) juntamente com CSS3 para melhor aparência do sistema. O objetivo do layout do site era ser simples ,de fácil compreensão e visualização.

Ao decorrer do desenvovimento do sistema, o que nos incomodou um pouco foi a forma em que o PHP interage com a pagina HTML, pois ao misturar os códigos das duas linguagens em uma única página (ou arquivo), ocorreu alguns equivocos que impediam o funcionamento correto do sistema.

Devido a disponibilidade dos autores, algumas funcionalidades importantes não foram implementadas, como:

  • Paginação
  • Login de Clientes
  • Responsividade funcionando parcialmente
  • Validação de campos de entrada

Outra dificuldade foi encontrada na tentativa de deixar a página responsiva, quando alguns elementos funcionavam corretamente na página no navegador do computador, ao diminuir a tela que exibe a página, os mesmos elementos se deslocavam sobre os outros.
